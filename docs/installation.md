# Installation Guide

## System Requirements

- Python 3.9 or higher
- pip package manager
- 2GB RAM minimum
- Internet connection (for WHOIS, DNS, SSL lookups)

## Linux / macOS

```bash
# Clone repository
git clone https://github.com/issu321/hackboard.git
cd hackboard

# Run installer
bash install.sh
```

## Windows

```powershell
# Clone repository
git clone https://github.com/issu321/hackboard.git
cd hackboard

# Install
pip install -e .

# Launch
hackboard
```

## pip Install

```bash
pip install hackboard
hackboard
```
