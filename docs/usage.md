# Usage Guide

## Launching HackBoard

```bash
# Default launch
hackboard

# Custom port
hackboard --port 8080

# Bind to all interfaces
hackboard --host 0.0.0.0

# Headless mode
hackboard --headless --no-browser
```

## Configuration Directory

```bash
hackboard --config-dir
# Output: /home/user/.config/hackboard
```
